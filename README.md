# kube-prometheus-go
# Requirements
- `Docker` installed
- `curl` installed
- `terraform` installed
- `kubectl` installed
- `helm` installed
- `kind` installed

# Setup Environment
- first:
```shell
  cd Environments/Terraform
```
- second: run the script to apply the terraform configurationScripts/terraform-apply.sh
```shell
    sh Scripts/terraform-apply.sh
```
